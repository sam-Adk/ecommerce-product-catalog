# 🛒 E-Commerce Product Catalog

A modern and responsive e-commerce product catalog web application that allows users to browse, search, and view products with a clean shopping interface.

---

## 📸 Preview

![Product Catalog](./screenshots/homepage.png)

---

## ✨ Features

- 🛍️ Product listing grid
- 🔍 Search and filter products
- 📦 Product detail view
- 💰 Price display system
- ❤️ Add to favorites (if implemented)
- 📱 Fully responsive design
- ⚡ Fast and lightweight UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local JSON data / API (optional)
